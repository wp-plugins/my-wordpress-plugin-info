== Description ==

This plugin recieves info of __any plugin__ from wordpress.org plugins repository. It may be useful for plugin developers and for bloggers who writes about plugins. Plugin uses Wordpress Plugins API, that was included to Wordpress since 2.7 version. See more details on [plugin homepage](http://simplelib.co.cc/?p=195).

Available languages:

  * English
  * Russian

If you have created your own language pack, or have an update of an existing one, you can send __.po__ and __.mo files__ to me so that I can bundle it into __My WP Plugin Info__.

== Installation ==

1. Upload plugin dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
